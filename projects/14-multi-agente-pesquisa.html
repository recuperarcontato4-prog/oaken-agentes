Multi-Agente Pesquisa

Três agentes colaborativos — Pesquisador, Crítico e Escritor — produzem relatórios completos com fontes verificadas.

Código destaque

from langgraph.graph import StateGraph, END
from langchain_community.tools import DuckDuckGoSearchResults

class ResearchState(TypedDict):
    topic: str
    sources: list[dict]
    draft: str
    feedback: str
    final_report: str

def researcher(state: ResearchState) -> ResearchState:
    """Pesquisa fontes via DuckDuckGo."""
    search = DuckDuckGoSearchResults(max_results=8)
    results = search.invoke(state["topic"])
    sources = parse_and_rank(results)
    return {"sources": sources}

def critic(state: ResearchState) -> ResearchState:
    """Avalia cobertura e factos do draft."""
    feedback = llm.invoke(CRITIC_PROMPT.format(
        draft=state["draft"],
        sources=state["sources"]
    ))
    return {"feedback": feedback}

def writer(state: ResearchState) -> ResearchState:
    """Gera relatório final em Markdown com citações."""
    report = llm.invoke(WRITER_PROMPT.format(
        sources=state["sources"],
        feedback=state["feedback"]
    ))
    return {"final_report": report}

O que aprendi

  • Colaboração multi-agente — Separar responsabilidades (pesquisa, crítica, escrita) melhora qualidade vs. agente monolítico.
  • Relatórios com fontes — Forçar citações inline reduz alucinações e aumenta confiança do leitor.
  • Supervisor pattern — Um nó coordenador decide a ordem de execução e quando terminar.
  • Search grounding — DuckDuckGo como fallback gratuito para web search em agentes experimentais.
