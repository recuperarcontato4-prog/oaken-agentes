LGPD Compliance Toolkit

API FastAPI para anonimizacao de dados pessoais com hash chain imutavel para auditoria LGPD.

FastAPI Hash Chain LGPD Anonimizacao

Arquitetura

  Raw Data (PII)
       |
  +----v-----------+
  |  Anonymizer    |
  |  API           |
  |  /anonymize    |
  +----+-----------+
       |
  +----v----+----v--------+
  | Anon    | Consent     |
  | Store   | Manager     |
  | (no PII)| (opt-in/out)|
  +---------+----+--------+
                  |
  +--------------v--------+
  |  Hash Chain Audit     |
  |  block N              |
  |  hash(N-1) + data     |
  |  = immutable trail    |
  +------------------------+

  Endpoints:
  POST /anonymize     - anonimiza dados
  POST /consent       - registra consentimento
  GET  /audit/{id}    - consulta trilha
  GET  /chain/verify  - verifica integridade

Codigo destaque

import hashlib, json
from fastapi import FastAPI
from pydantic import BaseModel
from datetime import datetime

app = FastAPI()

class AuditBlock(BaseModel):
    index: int
    timestamp: str
    action: str
    data_hash: str
    prev_hash: str
    block_hash: str

chain: list[AuditBlock] = []

def add_block(action: str, data: dict) -> AuditBlock:
    """Adiciona bloco a hash chain imutavel."""
    prev = chain[-1].block_hash if chain else "0" * 64
    data_hash = hashlib.sha256(
        json.dumps(data, sort_keys=True).encode()
    ).hexdigest()

    raw = f"{len(chain)}{prev}{data_hash}{action}"
    block_hash = hashlib.sha256(raw.encode()).hexdigest()

    block = AuditBlock(
        index=len(chain),
        timestamp=datetime.utcnow().isoformat(),
        action=action,
        data_hash=data_hash,
        prev_hash=prev,
        block_hash=block_hash,
    )
    chain.append(block)
    return block

@app.get("/chain/verify")
def verify_chain():
    """Verifica integridade de toda a hash chain."""
    for i in range(1, len(chain)):
        if chain[i].prev_hash != chain[i - 1].block_hash:
            return {"valid": False, "broken_at": i}
    return {"valid": True, "blocks": len(chain)}

O que aprendi

  • Hash chain para auditoria — encadear hashes garante que qualquer adulteracao e detectavel, sem blockchain completa.
  • Anonimizacao vs pseudonimizacao — LGPD distingue os dois; k-anonymity e o padrao minimo.
  • Consent management — registrar opt-in/out com timestamp e fundamental para compliance.
  • FastAPI para APIs de compliance — docs automaticos (Swagger) facilitam auditoria por DPO.
