Codigo destaque

import json, boto3

bedrock = boto3.client("bedrock-runtime", region_name="us-east-1")

def lambda_handler(event, context):
    """Lambda handler: recebe mensagem, chama Bedrock, retorna resposta."""
    body = json.loads(event["body"])
    user_msg = body["message"]

    response = bedrock.invoke_model(
        modelId="anthropic.claude-3-haiku-20240307-v1:0",
        contentType="application/json",
        body=json.dumps({
            "anthropic_version": "bedrock-2023-05-31",
            "max_tokens": 1024,
            "messages": [{
                "role": "user",
                "content": user_msg
            }]
        })
    )

    result = json.loads(response["body"].read())
    return {
        "statusCode": 200,
        "body": json.dumps({
            "reply": result["content"][0]["text"]
        })
    }

O que aprendi

  • Serverless para IA — Lambda + Bedrock escala de 0 a milhares de requests sem gerenciar servidores.
  • IaC com Terraform — infra como codigo garante reproducibilidade e versionamento do ambiente.
  • IAM least-privilege — policy minima para bedrock:InvokeModel reduz superficie de ataque.
  • Cold start mitigation — provisioned concurrency ou SnapStart minimiza latencia do primeiro request.
