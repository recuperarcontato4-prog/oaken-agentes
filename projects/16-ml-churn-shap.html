Código destaque

import xgboost as xgb
import shap
import streamlit as st

# Treinar modelo
model = xgb.XGBClassifier(
    n_estimators=200,
    max_depth=5,
    learning_rate=0.05,
    scale_pos_weight=ratio_neg_pos,
    eval_metric="aucpr",
)
model.fit(X_train, y_train, eval_set=[(X_val, y_val)],
          early_stopping_rounds=20)

# SHAP Explainer
explainer = shap.TreeExplainer(model)
shap_values = explainer.shap_values(X_val)

# Dashboard interativo

O que aprendi

  • Interpretabilidade — SHAP values transformam black-box ML em decisões auditáveis pelo negócio.
  • SHAP values — Force plots mostram a contribuição de cada feature para uma predição individual.
  • Simulação what-if — Sliders Streamlit permitem testar cenários e ver impacto em tempo real.
  • Class imbalance — scale_pos_weight e AUCPR são essenciais em datasets desbalanceados.
