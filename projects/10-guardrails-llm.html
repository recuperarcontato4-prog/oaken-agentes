Codigo destaque

import re, json, hashlib
from datetime import datetime

PII_PATTERNS = {
    "cpf": r"\d{3}\.?\d{3}\.?\d{3}-?\d{2}",
    "email": r"[\w.-]+@[\w.-]+\.\w+",
    "phone": r"\+?\d{2,3}[\s-]?\d{4,5}[\s-]?\d{4}",
}

def redact_pii(text: str) -> tuple[str, list[dict]]:
    """Remove PII e retorna texto limpo + log de redacoes."""
    findings = []
    for pii_type, pattern in PII_PATTERNS.items():
        for match in re.finditer(pattern, text):
            findings.append({
                "type": pii_type,
                "hash": hashlib.sha256(
                    match.group().encode()
                ).hexdigest()[:12],
            })
            text = text.replace(match.group(), f"[{pii_type.upper()}_REDACTED]")
    return text, findings

def audit_log(action: str, details: dict):
    """Append-only audit trail em JSONL."""
    entry = {
        "ts": datetime.utcnow().isoformat(),
        "action": action,
        **details,
    }
    with open("audit.jsonl", "a") as f:
        f.write(json.dumps(entry) + "\n")

O que aprendi

  • Defense in depth — PII redaction na entrada + toxicity filter na saida cria duas barreiras de seguranca.
  • LGPD compliance — hash do dado original permite auditoria sem armazenar PII em texto claro.
  • Append-only log — JSONL imutavel garante rastreabilidade completa de cada interacao.
  • Regex + LLM hybrid — regex pega padroes obvios; LLM classifier detecta PII contextual (nomes, enderecos).
