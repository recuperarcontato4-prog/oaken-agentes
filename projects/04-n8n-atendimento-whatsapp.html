Codigo destaque

from fastapi import FastAPI
from pydantic import BaseModel
from openai import OpenAI

app = FastAPI()
client = OpenAI()

class Message(BaseModel):
    phone: str
    text: str

INTENTS = ["agendamento", "duvida", "reclamacao", "outro"]

@app.post("/classify")
async def classify(msg: Message):
    """Classifica intencao da mensagem via LLM."""
    resp = client.chat.completions.create(
        model="gpt-4o-mini",
        messages=[{
            "role": "system",
            "content": f"Classifique a intencao em: {INTENTS}. Responda so a classe."
        }, {
            "role": "user",
            "content": msg.text
        }],
        temperature=0
    )
    intent = resp.choices[0].message.content.strip().lower()
    return {"phone": msg.phone, "intent": intent}

O que aprendi

  • n8n como orquestrador — workflows visuais aceleram prototipagem de automacoes complexas.
  • Classificacao de intencao — LLM com temperature=0 e few-shot atinge 95%+ de acerto em intents simples.
  • Evolution API vs Meta Cloud — Evolution e mais flexivel para multi-instancia e nao exige verificacao de negocio.
  • Webhook reliability — retry com idempotency key evita mensagens duplicadas.
