O que aprendi

  • State machines — Modelar fluxos de agente como grafos dirigidos com LangGraph simplifica debug e evolução.
  • Critic loop — Auto-avaliação com threshold impede respostas alucinadas de chegar ao utilizador.
  • Memória persistente — ChromaDB como vector store local elimina dependência de serviços cloud para RAG.
  • Convergência — Limitar iterações (max 3) evita loops infinitos sem sacrificar qualidade.
