Router Multi-Modelo

Router YAML que seleciona o melhor modelo LLM com base em politica custo vs qualidade, com fallback automatico.

Pydantic YAML Fallback Multi-LLM

Arquitetura

  Incoming Request
       |
  +----v-----------+
  |  Request       |
  |  Classifier    |
  |  (complexity)  |
  +----+-----------+
       |
  +----v-----------+
  |  YAML Policy   |
  |  Engine        |
  |  cost_weight   |
  |  quality_weight|
  +----+-----------+

Codigo destaque

from pydantic import BaseModel
import yaml

class ModelSpec(BaseModel):
    name: str
    cost_per_1k: float
    quality_score: float
    max_tokens: int

class RoutingPolicy(BaseModel):
    cost_weight: float = 0.4
    quality_weight: float = 0.6
    models: list[ModelSpec]
    fallback_order: list[str]

def select_model(policy: RoutingPolicy, complexity: str) -> ModelSpec:
    """Seleciona modelo com melhor score ponderado."""
    def score(m: ModelSpec) -> float:
        cost_norm = 1 - (m.cost_per_1k / max(
            x.cost_per_1k for x in policy.models
        ))
        return (
            policy.cost_weight * cost_norm +
            policy.quality_weight * m.quality_score
        )

    if complexity == "high":
        # Forcar modelo top-quality
        return max(policy.models, key=lambda m: m.quality_score)

    return max(policy.models, key=score)

def route_with_fallback(policy: RoutingPolicy, prompt: str) -> str:
    """Tenta modelo ideal, fallback em caso de erro."""
    best = select_model(policy, classify_complexity(prompt))
    try:
        return call_model(best.name, prompt)
    except Exception:
        for name in policy.fallback_order:
            if name != best.name:
                return call_model(name, prompt)

O que aprendi

  • Routing inteligente — nem toda pergunta precisa do modelo mais caro; 80% das queries sao simples.
  • Politica como YAML — separar regras de negocio do codigo permite ajustar sem redeploy.
  • Fallback chain — resiliencia multi-provider garante 99.9%+ de uptime efetivo.
  • Score ponderado — weighted sum de custo e qualidade e simples e surpreendentemente eficaz.
