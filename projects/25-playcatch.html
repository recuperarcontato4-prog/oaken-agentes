PlayCatch — DJ Musical com IA

DJ inteligente que analisa sentimentos com BERT, recomenda músicas com feedback loop e conversa via chatbot zero-shot — tudo em Gradio.

Código destaque

from transformers import pipeline
import gradio as gr

# Análise de sentimentos com BERT
sentiment = pipeline(
    "sentiment-analysis",
    model="nlptown/bert-base-multilingual-uncased-sentiment",
)

# Chatbot zero-shot para perguntas musicais
chatbot = pipeline(
    "zero-shot-classification",
    model="facebook/bart-large-mnli",
)

def analyze_mood(text: str) -> dict:
    """Detecta mood do utilizador via BERT."""
    result = sentiment(text)[0]
    stars = int(result["label"][0])  # "4 stars" → 4
    mood_map = {1: "melancólico", 2: "calmo",
                3: "neutro", 4: "animado", 5: "eufórico"}
    return {"mood": mood_map[stars], "score": result["score"]}

def recommend(mood: str, feedback: list) -> list:
    """Recomenda músicas ajustando por feedback anterior."""
    base = catalog.query(mood=mood, limit=10)
    # Boost/penalize baseado em thumbs up/down
    for song in base:
        song.weight += feedback_score(song.id, feedback)
    return sorted(base, key=lambda s: s.weight, reverse=True)[:5]

# Interface Gradio multi-tab
with gr.Blocks(theme=gr.themes.Soft()) as app:
    with gr.Tab("Mood DJ"):
        txt = gr.Textbox(label="Como você está?")
        btn = gr.Button("Analisar")
        out = gr.JSON(label="Resultado")
        btn.click(analyze_mood, txt, out)

O que aprendi

  • Análise sentimentos BERT — Modelo multilingual detecta mood em português sem fine-tuning adicional.
  • Recomendação com feedback — Thumbs up/down ajustam pesos em tempo real, personalizando resultados.
  • Chatbot zero-shot — BART-MNLI responde perguntas sobre música sem dados de treino específicos.
  • Gradio multi-tab — Interface rica com tabs, chatbot e controles interactivos em poucas linhas.
