Bank Classifier

Zero-Shot Classification de transações bancárias com mDeBERTa — categoriza gastos sem dados de treino rotulados.

HuggingFace Transformers Zero-Shot NLI

Código destaque

from transformers import pipeline
import pandas as pd

# Pipeline zero-shot com mDeBERTa multilingual
classifier = pipeline(
    "zero-shot-classification",
    model="MoritzLaurer/mDeBERTa-v3-base-mnli-xnli",
    device=0,  # GPU
)

# Categorias sem dados de treino
categories = [
    "alimentação", "transporte", "moradia",
    "saúde", "lazer", "educação",
    "transferência", "salário", "investimento",
]

def classify_transactions(df: pd.DataFrame) -> pd.DataFrame:
    """Classifica transações via NLI zero-shot."""
    results = classifier(
        df["description"].tolist(),
        candidate_labels=categories,
        hypothesis_template="Esta transação é de {}.",
        batch_size=32,
    )
    df["category"] = [r["labels"][0] for r in results]
    df["confidence"] = [r["scores"][0] for r in results]
    return df

# Processar e exportar
txns = pd.read_csv("extrato.csv")
classified = classify_transactions(txns)
classified.to_csv("categorized.csv", index=False)

O que aprendi

  • NLI para classificação — Natural Language Inference permite classificar sem dataset rotulado, só com labels textuais.
  • Zero-shot — Adicionar/remover categorias é instantâneo, sem re-treino do modelo.
  • mDeBERTa multilingual — Suporta português nativamente com performance próxima de modelos monolíngues.
  • Hypothesis template — Template bem formulado impacta diretamente a precisão da classificação.
