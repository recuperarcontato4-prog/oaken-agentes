Código destaque

import torch
import torchvision.models as models
from torchvision import transforms

# Transfer learning: ResNet18 pré-treinada
model = models.resnet18(weights=models.ResNet18_Weights.DEFAULT)

# Congelar backbone
for param in model.parameters():
    param.requires_grad = False

# Substituir classifier final
model.fc = torch.nn.Sequential(
    torch.nn.Linear(512, 256),
    torch.nn.ReLU(),
    torch.nn.Dropout(0.3),
    torch.nn.Linear(256, num_classes),
)

# Treinar apenas novas camadas
optimizer = torch.optim.Adam(model.fc.parameters(), lr=1e-3)
criterion = torch.nn.CrossEntropyLoss()

for epoch in range(10):
    for images, labels in train_loader:
        outputs = model(images)
        loss = criterion(outputs, labels)
        loss.backward()
        optimizer.step()
        optimizer.zero_grad()

# Export TorchScript para produção
scripted = torch.jit.script(model.eval())
scripted.save("model.pt")

O que aprendi

  • Transfer learning — Reutilizar features de ImageNet reduz tempo de treino de horas para minutos.
  • Freeze + fine-tune — Congelar backbone e treinar só o classifier evita catastrophic forgetting.
  • TorchScript export — Modelo serializado roda em C++ sem dependência de Python em produção.
  • Data augmentation — RandomHorizontalFlip + ColorJitter aumentam robustez com dataset pequeno.
